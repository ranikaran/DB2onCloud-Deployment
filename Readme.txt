1. Unzip the file in a directory
2. Configure the class path with "db2jcc4.jar" and "jsch-0147.jar"
3. Update "config.properties" file with server configuration parameters ( it is self explanatory) and set the log file name as well
4. run the file from the same directory where you extracted the zip
	c:\> java DB2OnCloudTest
	
5. Look at the log file for the test results
